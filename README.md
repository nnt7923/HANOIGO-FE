# HanoiGo Frontend

React + TypeScript + Vite frontend for HanoiGo.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Default local app URL:

```text
http://localhost:5173
```

Default backend API:

```text
http://localhost:5000/api
```

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
